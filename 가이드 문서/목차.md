## 빠른 접근

### 처음 합류한 팀원이라면
1. `프로젝트_총괄/프로젝트_개요.md` — 전체 큰 그림 파악
2. `프로젝트_총괄/빌드_실행.md` — 빌드 환경 세팅
3. `팀_관리/인수인계.md` — 함정과 주의사항 숙지

### NPC/트레이너를 추가하고 싶다면
→ `NPC_트레이너/NPC_추가_가이드.md`

### 새 시스템/기능을 구현하고 싶다면
→ `프로젝트_총괄/코드_구조.md` → 관련 `시스템_구현/` 문서

### 최근 무슨 작업이 됐는지 확인하려면
→ `팀_관리/변경_이력.md`

### 아이템 시스템을 구현하고 싶다면
→ `시스템_구현/아이템_연동.md`

## 핵심 소스 파일 바로가기

```
src/data/map_data.h      NPC/트레이너/표지판/워프 데이터
src/game/game.cpp        Scene 상태머신 (메뉴 포함)
src/game/overworld.cpp   이동/충돌/NPC/표지판 상호작용
src/game/battle.cpp      턴제 배틀 엔진
src/game/player.h        플레이어 상태 + 스토리 플래그
src/data/pokemon_data.h  포켓몬 종족/기술 데이터
src/data/type_chart.h    타입 상성표
build.bat                빌드 + 실행
```
